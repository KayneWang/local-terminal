# Local Terminal

> 为 xterm 打造本地终端模拟功能

看过文档之后会发现直接使用 xterm 模拟本地终端十分困难，官方 demo 也很简单，不能满足复杂需求。

所以，该项目希望提供一套常用的终端操作，帮助快速模拟本地终端。
